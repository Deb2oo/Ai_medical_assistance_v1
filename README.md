## 💡 **Features**  
✅ AI-driven **medical chatbot** using OpenAI GPT/BERT  
✅ **Facial expression analysis** (YOLOv8 & OpenCV)  
✅ **Speech-to-text processing** (FFmpeg & PortAudio)  
✅ **User-friendly Gradio UI**  
✅ **Secure API integration** for real-time diagnosis
